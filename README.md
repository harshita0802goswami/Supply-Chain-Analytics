##  Project Objectives

* Monitor key supply chain service-level KPIs on a daily basis.
* Compare actual performance against customer-specific targets.
* Identify customers, cities, and product categories with poor delivery performance.
* Perform root cause analysis to determine whether failures are due to late deliveries, incomplete deliveries, or both.
* Enable data-driven decision-making through interactive visualizations.

## Key Insights

* OTIF performance remained significantly below the target, indicating opportunities to improve service levels.
* Several customers consistently experienced poor delivery performance due to delayed and incomplete shipments.
* Daily products contributed the highest number of failed order lines, impacting overall OTIF.
* Service levels varied across cities, highlighting location-specific operational challenges.
* Weekly trend analysis enabled continuous monitoring of supply chain performance and supported proactive decision-making.
